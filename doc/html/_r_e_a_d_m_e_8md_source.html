20 ### Shortest Path algorithm
21 The folder ShortestPath contains an implementation of the [Dijkstra algorithm](http://en.wikipedia.org/wiki/Dijkstra%27s_algorithm), used to find the shortest path in a weighted graph. It reads a file with two strings: node 1, and node 2, separated by a blank space between them, and followed by a number; the weight of going from node 1 from node 2. The graph gets saved in an adjacency matrix, which is not at all memory efficient, but was selected as a visualization method for learning the algorithm.
